# Pooja Pansare - Portfolio (Vite + React + Tailwind) - Tech Blue Edition

## Quick start

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Deploy to Vercel

1. Create a GitHub repo and push this project.
2. Sign up at https://vercel.com and import the repo.
3. Vercel will detect the project (Vite) and run `npm run build`.

## Notes
- Accent color: Tech Blue (#007BFF)
- Place your resume `/public/Pooja_Pansare_Modern_Style_Resume.docx` to enable the Download Resume button.
- Photo placeholder is in `src/assets/photo-placeholder.png`
